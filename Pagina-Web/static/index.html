Resumo sobre mim:

Sou uma estudante universitária do curso de Engenharia de Software na universidade federal de Goiás (UFG).

Meu interesse por programação surgiu em 2022, quando eu fiz um curso de lógica de programação no Senac.

Conhecimentos na área:

GoLang

GoLang, ou Go, é a primeira linguagem de programação que comecei a estudar.

Conheci essa linguagem na disciplina de Introdução a Programação e tenho estudado, não só para a matéria, mas também para minha vida profissional futura. A linguagem em questão tem sido usada nas atividades da disciplina para a criação de algoritmos simples, mas também será usada para a execução do projeto integrador, cujo foco é o rastreamento do câncer de colo do útero.

C++

C++ é uma linguagem que comecei a estudar em uma disciplina núcleo livre chamada "Meninas na Programação".

É uma linguagem usada na programação competitiva, um nicho que despertou meu interesse recentemente. Além de usar essa linguagem nas aulas dessa disciplina, também utilizei na "maratona feminina de programação" (MFP).

Alguns projetos

Harumi Tokushima

Um projeto pessoal que eu gostei muito de fazer envolve a criação de uma personagem original (OC, so iglês "Original Character").

O design foi feito no Figma e permite ver os diferentes trajes da personagem, além de uma descrição dela. Em resumo, está personagem foi baseada no universo do mangá (e anime) "Boku no Hero Academia" (também conhecido pelo nome em inglês "My Hero Academia")
